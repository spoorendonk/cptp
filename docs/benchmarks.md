## Summary

| Set | Instances | Solved | Solve rate |
|---|--:|--:|--:|
| SPPRCLIB | 45 | 45 | 100% |
| Roberti Set 3 | 31 | 26 | 84% |
| **Total** | **76** | **71** | **93%** |

Cut types separated:
- **SEC** -- Generalized subtour elimination constraints via Gomory-Hu tree min-cuts
- **RCI** -- Rounded capacity inequalities (demand-based rounding of capacity cuts)
- **Multistar** -- GLM inequalities (generalized large multistar)

RGLM (rounded GLM) and Comb separators are available but not enabled by default.

## Reproducing

```bash
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build -j$(nproc)

# Single instance
./build/rcspp-solve bench/instances/spprclib/B-n45-k6-54.sppcc --time_limit 60

# Roberti instance
./build/rcspp-solve bench/instances/roberti/E-n76-k8_b.vrp --time_limit 300
```

Detailed per-instance statistics (LP iterations, timing breakdown, per-separator cut counts) are in [`bench/progress.csv`](../bench/progress.csv).
